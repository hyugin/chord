Chord is ad-hoc signed (no Apple Developer ID).
Every release DMG is a new binary hash, so macOS quarantines the download and shows a malware warning if you open Chord (or any .command helper) directly.

From Terminal, in the folder with the DMG (replace the version):

DMG="chord-<version>.dmg"
hdiutil attach "$DMG" -nobrowse -quiet
cp -R "/Volumes/Chord/Chord.app" /Applications/
xattr -cr /Applications/Chord.app
open /Applications/Chord.app
hdiutil detach "/Volumes/Chord" -quiet

Or drag Chord.app onto the Applications shortcut in this window, then:

xattr -cr /Applications/Chord.app && open /Applications/Chord.app

That clears the quarantine flag and launches Chord.
Repeat after each new download.

Building from source also skips Gatekeeper — see the project README.
